Administrator Privileges Required

Một số lệnh cấu hình mạng hệ thống (như đổi DNS hoặc MTU) yêu cầu chạy máy chủ Python với quyền Admin để thực thi trực tiếp.

Bạn có thể tự tay thực thi lệnh này bằng cách mở PowerShell (Admin) và nhập:

Set-DnsClientServerAddress -InterfaceAlias Wi-Fi -ServerAddresses ("1.1.1.1", "8.8.8.8")
Cách khởi chạy Backend với quyền Administrator:
  1. Đóng cửa sổ dòng lệnh Python server hiện tại.
  2. Tìm và mở ứng dụng Command Prompt hoặc PowerShell bằng cách nhấp chuột phải chọn "Run as Administrator".
  3. Chạy lại lệnh: python server.py để cho phép Dashboard áp dụng cấu hình tự động chỉ với 1-click.
